Implementation of NearestNeighbor algorithm in prediting ratings for users

This project implements the K-Nearest neighbor algorithm in which an object is classified by majority vote of its neighbors, with the object being assigned to the most common class among itsÂ  k-nearest neighbors.
The performance of this algorithm has been tested on real-world data related to books and their ratings.
The project focuses on sampling existing data which will help to recommend books for new users
